# AI GTM Copilot – Forward Deployed Engineering Demo

# A lightweight AI-native CRM workflow prototype demonstrating how Forward Deployed Engineers can build customer-facing AI solutions for GTM teams.

This project demonstrates how a Forward Deployed Engineer can rapidly prototype AI-powered GTM workflows for customers.

## Problem
GTM teams often struggle with:
- fragmented CRM workflows
- manual lead qualification
- slow outreach preparation
- poor visibility across sales tools

## Solution
This prototype simulates an AI-native CRM assistant that:

1. Ingests lead/company data
2. Generates an AI account brief
3. Creates personalized outreach
4. Suggests implementation steps
5. Stores leads in a CRM-style database

## Features

- AI account research
- automated outreach generation
- lead scoring
- workflow recommendations
- CRM lead storage
- exportable CRM dataset

## Tech Stack

Python  
Streamlit  
OpenAI API  
SQLite  
Pandas

## Architecture

Lead Input → AI Analysis → Outreach Generation → CRM Storage

## Why This Project

Forward Deployed Engineers build custom workflows that help customers operationalize AI.

This demo simulates that process by showing how AI can be integrated directly into GTM workflows to accelerate sales execution.

## Demo Flow

1. Enter company + lead details
2. Generate AI account brief
3. Generate outreach email
4. Save lead to CRM
5. View CRM analytics
